Friend Controller Documentation

Overview

This controller manages user friendship operations such as sending, accepting, and rejecting friend requests, displaying friends, and removing them.

Functions

Get_Users(req, res)

Retrieves users that can be added as friends.

  • Extracts user ID from the request
  • Calls findUsers(userid)
  • Returns potential friends or error

Send_FriendReq(req, res)

Sends a friend request to another user.

  • Sender ID from request
  • Target ID from body
  • Calls addFriend(user1id, user2id)

Show_FriendReq(req, res)

Retrieves pending friend requests.

  • User ID from request
  • Calls showFriendreq(userid)
  • Returns friend requests or error

Accept_FriendReq(req, res)

Accepts a pending friend request.

  • Recipient ID from request
  • Requester ID from body
  • Calls acceptFriend(user1id, user2id)

Reject_FriendReq(req, res)

Rejects a pending friend request.

  • Recipient ID from request
  • Requester ID from body
  • Calls rejectFriend(user1id, user2id)

Fetch_Friends(req, res)

Retrieves the list of current friends.

  • User ID from request
  • Calls displayFriend(userid)

Remove_Friend(req, res)

Removes a user from the friend list.

  • User ID from request
  • Friend ID from body
  • Calls removeFriend(user1id, user2id)

Dependencies

  • findUsers - Get potential friends
  • addFriend - Send friend request
  • showFriendreq - Get pending requests
  • acceptFriend - Accept request
  • rejectFriend - Reject request
  • displayFriend - List of friends
  • removeFriend - Remove a friend

Error Handling

  • All methods use try-catch blocks
  • Returns HTTP 500 with descriptive message on error
  • Logs include relevant user IDs for tracing

Export

This controller exports all seven friend management functions listed above.
