About the Project

Battle of Quiz Lords is a dynamic and interactive quiz-based web application designed to provide users with a competitive yet fun learning experience. The platform enables users to engage in various quiz modes, supported by a robust, scalable backend infrastructure and modern DevOps practices.

Key Features

  • Solo Match
  • Multiplayer Match
  • Play Challenges
  • Create Challenges
  • Profile Management
  • Leaderboard
  • Friends System
  • Feedback Submission

System Architecture and Implementation

Frontend

  • Developed using Vite React, offering fast development and optimized builds.
  • Fully functional and deployed inside a Docker container for efficient delivery and scaling.

Backend

  • Developed and fully functional using Node.js and Express.js.
  • Handles all logic and API interactions.
  • Dockerized and deployed for easy scaling and deployment.

Database

  • Uses Supabase as a backend-as-a-service platform.
  • Built on PostgreSQL and serves as an open-source alternative to Firebase.
  • Chosen for its free tier, easy setup, and developer-friendly environment.

DevOps and Site Reliability Engineering (SRE)

Containerization

  • Both frontend and backend are containerized using Docker.
  • Docker images are created and deployed efficiently.

Kubernetes

  • Used for managing Pods, Services, and Deployments.
  • Includes Horizontal Pod Autoscaling for dynamic load handling.

Monitoring and Observability

  • Prometheus: Collects performance metrics and monitors server health.
  • Grafana: Visualizes metrics with real-time dashboards, including DB monitoring.
  • Loki: Centralized logging for debugging and tracking application events.

CI/CD

  • Jenkins: Automates deployment process for the application.

Why We Chose This Project

As a team of 7 passionate developers, we wanted to go beyond traditional project templates and build something from scratch—an application that challenged us technically, creatively, and collaboratively.

Our core motivation was to understand the end-to-end development process:

  • From planning and UI/UX design
  • To backend logic, APIs, and database architecture
  • To real-world issues like user management, matchmaking, scaling, and monitoring

We brainstormed many ideas but ultimately chose Battle of Quiz Lords because it offered the perfect blend of fun, competition, and learning. The competitive aspect makes it feel like a game while still being educational. It supports both casual play and serious competition, making it appealing to a broad audience.

The technical challenges — such as real-time multiplayer matchmaking, scoring, and system monitoring — gave us real insight into building a production-ready app. It allowed us to sharpen our full-stack skills and experience the full journey of software development.
